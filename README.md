# 2021-06-26  企业信息查询工具
### 原理：爬取企信通信息
```
用法 python qixinton.py 企业名称
使用时，会弹出二维码，微信扫码登录qxt
```
### 获取的信息包括
* 企业邮箱
* 企业电话
* 企业分支机构
* 企业的对外投资情况
